## 写在前面

处理页面需求时，通常需要快速制作页面。而页面布局往往是起决定性作用，一个好的布局不仅能保证结构优美的HTML，而且能加快开发速度。这里对平时的常用布局进行一个汇总。

### Flex 弹性布局

> 相对传统布局，flex布局可以简便、完整、响应式地实现各种页面布局。这里主要依托flex实现各种需求，再针对兼容性做优雅降级处理。
